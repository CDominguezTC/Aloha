8.3 Comunicación

De clic en [Sistema] > [Comunicación] > [Comando de Dispositivos], se mostrara la lista de comandos.

 

Si el valor regresado es mayor o igual a 0, el comando fue emitido exitosamente. Si el comando regresado es menor de 0, el comando no pudo ser emitido.

Borrar Comandos: Limpia la lista de comandos.

Exportar: Exporta la lista de comandos.
