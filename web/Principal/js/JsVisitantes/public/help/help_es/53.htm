4.4.1 Zona

Para el uso de las funciones avanzadas de control de acceso como el Anti-Passback Global es necesario definir las zonas de acceso.

l  Agregar

1. De clic en [Control de Acceso] > [Funciones Avanzadas] > [Zona] > [Nuevo] para acceder a la interfaz de Agregar Zona:

2. Ingrese el ID de Zona, Nombre y Notas si es necesario.

3. De clic en [OK] para guardar y salir. La zona agregada se mostrará en la lista.
