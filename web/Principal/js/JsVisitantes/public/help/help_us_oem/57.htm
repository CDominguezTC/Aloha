4.4.5 Global Linkage

The global linkage function allows you to configure data across devices. Only push devices support this function.

l  Add

1. Click [Access Control] > [Access Rule]> [Advanced Functions] > [Global Linkage] > [New]:

2. Choose Global Linkage trigger conditions, the input point (System will filter devices according to the choice in first step) and the output point, Set up linkage action. More details about these parameters, please refer to 4.3.2 Linkage Setting.

?Note: You can select multiple Door Events, but “Fail to connect server”, “Recover connection”, “Device connection off” will be filtered automatically from Door Event.

2. Click [OK] to save and quit. The added Global Linkage will display in the list.
