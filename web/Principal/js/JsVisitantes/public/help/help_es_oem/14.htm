2.5 Modificar Contraseña

Puede modificar la contraseña de inicio de sesión en [Información Personal] :

Marque la casilla [Restablecer Contraseña] para modificar la contraseña.

Nota: El súper usuario y el usuario nuevo creado por el súper usuario tienen la contraseña 111111 por default. El nombre de usuario es sensible a mayúsculas y minúsculas.
