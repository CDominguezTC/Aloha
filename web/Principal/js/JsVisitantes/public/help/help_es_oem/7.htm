1.5 Breve Introducción de Módulo de Visitantes

El módulo de visitantes es un sistema de gestión basado en web que implementa el registro de entrada, registro de salida, captura de imágenes, brinda estadísticas de la cantidad de visitantes y permite gestionar las reservaciones, así como compartir información entre los sitios de registro. Está integrado con el sistema de control de acceso y sistema de control de elevador, generalmente utilizado en mostradores para recepción y puertas de empresas, para administrar y gestionar visitantes.
