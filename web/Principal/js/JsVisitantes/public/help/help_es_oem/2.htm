1 Introducción del Sistema

La gestión de seguridad en se ha vuelto cada vez más importante para empresas y corporativos. El manejo y dominio de éste sistema ayuda a los usuarios a integrar las operaciones y procedimientos de seguridad en una sola plataforma. El sistema está dividido en seis módulos: personal, acceso, elevador, visitantes, video y sistema.

                Características del Sistema

m  Procesamiento de gran capacidad de datos, permitiendo la gestión de hasta 30,000 usuarios.

m  Gestión de privilegios multi-nivel asegura la confidencialidad de los datos.

m  Sistema de obtención de datos en tiempo real que asegura la pronta retroalimentación del administrador.

                Requerimientos de PC

m  CPU: 2 Núcleos o superior.

m  Memoria: 4G o superior.

m  Hardware: Espacio disponible de 10G o superior. Es recomendado usar un disco NTFS en la partición del disco donde se instalará el software.

m  Resolución del monitor: 1024 x 768 pixeles o superior.

                Sistema Operativo

m  Sistemas operativos soportados: Windows 7/Windows 8/Windows 8.1/Windows Server 2008 (32/64).

m  Base de datos default: PostgreSQL

m  Navegadores recomendados: IE 11+/Firefox 27+/Chrome 33+

?Nota: Para el registro de huella digital debe usar IE 8.0 o superior.
